## 프로젝트 제작동기

- 디자인 부터 코딩까지 100% 혼자 작업한 프로젝트 입니다.
- 디자인은 그리드 시스템을 사용하였습니다.
- html5,css3, 바닐라 자바스크립트를 사용해서 구현한 웹사이트입니다.
- 반응형 페이지로 구현 하였습니다.

## 기술스택

- 디자인
  - photoshopcc2021, illustratorcc2021, figma
- 코딩
  - html5
  - css3
  - es6
